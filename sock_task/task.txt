The scocket is a tcp/ip client,
ssocket is a server.
Client connects to the server and whatever the client or server types in the console,
it is sent to the other party and displayed on the respective console.
Server accepts only one connection.

Task 1)
Make the ssocket an echo server, so it sends back what was sent to him.

Task 2)
Make the ssocket accept two connection and route data between them,
working as a chat server for the two clients.
Whatever client1 types is displayed in client2 and the other way round.
